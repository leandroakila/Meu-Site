# 🌐 LN Sistemas - Site Institucional

**Slogan:** Soluções inteligentes em sistemas e automação de processos

---

## 📋 Sobre o Projeto

Site institucional profissional desenvolvido para a LN Sistemas, empresa especializada em desenvolvimento de sistemas personalizados, automação de processos e soluções digitais para empresas e profissionais.

O site foi criado para apresentar os serviços, soluções e experiência da empresa, com foco em atrair clientes que necessitam de tecnologia para organizar informações, automatizar tarefas e melhorar a gestão de dados.

---

## ⚙️ Como Usar

### Instalação
1. Faça o download de todos os arquivos
2. Mantenha a estrutura de pastas intacta
3. Abra `index.html` em qualquer navegador moderno

### Configurações Necessárias

#### WhatsApp
No arquivo `js/main.js`, atualize o número do WhatsApp (linha ~98):
```javascript
const whatsappNumber = '5562999999999'; // Substitua pelo número real
```

#### Email
Atualize o email em todas as páginas se necessário:
- Busque por: `contato@lnsistemas.com`
- Substitua pelo email real

---

## 🔧 Customização

### Cores
Para alterar as cores do site, edite as variáveis CSS em `css/style.css`:
```css
:root {
    --color-primary: #0A2540;
    --color-secondary: #1E90FF;
    /* ... outras cores */
}
```

### Conteúdo
Todos os textos podem ser editados diretamente nos arquivos HTML.

### Imagens
O site usa ícones do Font Awesome. Para adicionar imagens reais:
1. Crie uma pasta `images/`
2. Adicione suas imagens
3. Substitua os ícones por tags `<img>`
